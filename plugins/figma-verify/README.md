# figma-verify

Figma 디자인 기반 프론트엔드 마크업 작업 후, 구현 결과물을 자동 검증하고 수정하는 워크플로우 플러그인.

## 사용법

마크업 작업 완료 후 다음 명령어를 실행:

```
/figma-verify:verify
```

## 워크플로우

1. Figma 원본 디자인 스크린샷 확보
2. `agent-browser`로 구현 결과물 스크린샷 촬영
3. 두 스크린샷 비교하여 차이점 식별
4. 차이점 수정 후 재검증
5. 완료 보고

## 요구사항

- `agent-browser` 스킬이 설치되어 있어야 합니다
- Figma MCP 서버가 연결되어 있어야 합니다
- 개발 서버가 실행 중이어야 합니다

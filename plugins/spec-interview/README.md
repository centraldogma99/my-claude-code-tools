# spec-interview

심층 인터뷰를 통해 기능 사양서(spec)를 작성하는 플러그인입니다. `AskUserQuestion` 도구를 활용하여 기술적 구현, UI/UX, 트레이드오프 등을 깊이 있게 질문하고, 인터뷰 결과를 기반으로 사양서를 자동 생성합니다.

## 사용법

```
/interview 회원가입 플로우 리뉴얼
/interview 지도 배정 워크숍 MVP
/interview 인앱 결제 시스템 도입
```

## 동작

1. 입력한 주제에 대해 심층 인터뷰 시작
2. 기술적 구현, UI/UX, 우려 사항, 트레이드오프 등을 반복 질문
3. 인터뷰 완료 후 사양서를 파일로 저장

## 출력 파일

저장 경로를 별도로 지정하지 않으면 프로젝트 루트에 `spec-<주제를-kebab-case로>.md` 형태로 저장됩니다.
